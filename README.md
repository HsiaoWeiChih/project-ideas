# Project Ideas

## 🏸 羽球排場系統
**狀態**：Idea stage（尚未實作）  
**目標**：自動化羽球排場、配對與積分管理，減少人工安排的混亂

---

### 🎯 核心功能

#### 1. 配對與排場
- 隨機分配下場組合  
- 每場結束後可記錄勝負並累積積分  
- 依據積分動態調整對手  
  - 打越多場，配到的對手實力越接近  

#### 2. 綁定 / 不綁定機制
- 可指定固定搭檔  
  - 設定後，後續每一場都與該人配對  
- 可指定「不與某人配對」  
  - 之後的隨機配對中不會再配到該人  

#### 3. 區域管理
- 備戰區  
- 休息區  

---

### 🔊 廣播與娛樂功能
- 廣播系統  
  - 可連接藍牙喇叭  
  - 自動語音廣播下場組合  
- Spotify 串接  
  - 使用者可登入並播放音樂  

---

### 🔐 權限控管
- 綁定 / 不綁定設定權限  
- QR Code 登入控制：
  - 進入備戰區與休息區  
  - 僅可查看資訊（不可操作控制功能）
